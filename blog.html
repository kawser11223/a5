Question-1: What are the different ways to select an element in the DOM?

JavaScript-এ DOM Document Object Model থেকে এলিমেন্ট সিলেক্ট করার জন্য
নির্দিষ্ট id দেওয়া এলিমেন্ট সিলেক্ট করতে ব্যবহৃত হয়।
একই ক্লাসের সব এলিমেন্ট সিলেক্ট করতে ব্যবহৃত হয়।
একটি HTMLCollection রিটার্ন করে যা অ্যারের মতো কিন্তু পুরোপুরি অ্যারে নয়।
CSS সিলেক্টরের মতো স্টাইল ব্যবহার করে সব মিলছে এমন এলিমেন্টগুলোর লিস্ট NodeList রিটার্ন করে।
CSS সিলেক্টরের মতো স্টাইল ব্যবহার করে প্রথম ম্যাচিং এলিমেন্টটি সিলেক্ট করে।

Question-2: What is the difference between innerHTML, innerText, and textContent ?

যদি কোনো HTML ট্যাগ থাকে, তাহলে innerHTML সেট করলে সেই HTML প্রসেস হয় এবং ব্রাউজারে রেন্ডার হয়।
এটি HTML ট্যাগ ইগনোর করে সমস্ত টেক্সট রিটার্ন করে, এমনকি display: none থাকলেও

Question-3: What is event delegation in the DOM?

যখন একটি ইভেন্ট ঘটে, তখন সেটি Event Bubbling এর মাধ্যমে parent element পর্যন্ত পৌঁছে যায়। আমরা যদি parent element-এ ইভেন্ট লিসেনার যুক্ত করি, তাহলে সেটি child elements-এর ইভেন্টও ধরতে পারবে।

Question-4: What is event bubbling in the DOM?

একটি ছোট এলিমেন্টে ইভেন্ট হলে, সেটি তার বড় parent এলিমেন্টগুলোতেও পৌঁছে যায়।
